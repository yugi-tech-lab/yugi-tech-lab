# Yugi Tech Lab（ユギ・テックラボ）🔧✨

遊んで学んでMakerになろう！

## 活動内容
- Maker Fairや、つくろXX等、工作展示イベントへの出展
- 作品コンテストへのチャレンジ
- 子ども向け・初心者向けの工作ワークショップ企画
- ショップでのキット販売(予定)
- Youtubeでの発信(これから頑張る)

## 主な作品
- 🧩 **BreadMaker**：ブレッドボードを使った初心者向けの作品、ワークショップ
- 🎮 **改造玩具**：野球盤など、従来のおもちゃを電子工作で改造
- 💡 **便利ツール**：電子工作で役に立つアプリやハードの提供
